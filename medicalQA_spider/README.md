# 医疗问答信息爬虫
## 描述
  基于requests和xpath爬取医疗问答网站的相关信息，并且使用了fake-useragent伪装了请求头，最后结果保存至MongoDB数据库中。
## 软件环境
*python >= 3;\
*requests;\
*xpath;\
*fake-useragent;\
*mongodb;\
